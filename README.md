# Connect4 MinMax AI

## 概要

Connect4（四目並べ）に対して、MinMax法を用いたAIを実装したプロジェクトです。

評価関数のパラメータが異なる2種類のAIを作成し、100試合の自己対戦を行うことで評価関数の違いが勝率に与える影響を比較しました。

また、各試合の対戦履歴や評価値をログとして保存し、AIの行動や対局結果を分析できるようにしています。

---

## 主な機能

- Connect4のゲームロジック実装
- MinMax法による探索AI
- 評価関数による局面評価
- AI同士の自己対戦
- 対戦ログの自動保存
- 各手の評価値の出力
- 先攻・後攻を交互に変更した公平な比較

---

## 評価関数

評価関数では、盤面上の連結状態を評価して局面の良し悪しを数値化しています。

以下の要素を評価対象としています。

- 2連結
- 3連結

連結数ごとの評価値を変更することで、異なる戦略を持つAIを作成できます。

---

## 実験内容

以下の2種類のAIを用いて100試合の自己対戦を実施しました。

### Three-focus AI

- 2連結：10点
- 3連結：50点

3連結の形成を重視するAIです。

### Two-focus AI

- 2連結：30点
- 3連結：40点

2連結の形成も積極的に評価するAIです。

先攻有利の影響を軽減するため、各試合ごとに先攻・後攻を交代させています。

実験の結果、3連結を重視する戦略だけでなく、2連結を積極的に評価する戦略も有効であることが確認できました。
