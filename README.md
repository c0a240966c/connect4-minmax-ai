# Connect4 MinMax AI

## 概要

Connect4（四目並べ）に対して、MinMax法を用いたAIを実装したプロジェクトです。

評価関数のパラメータを変更した2種類のAIを作成し、自己対戦による性能比較を行いました。対戦結果はログファイルとして保存され、評価関数の違いが勝率に与える影響を分析できます。

## 主な機能

* Connect4のゲームロジック実装
* MinMax法による手の探索
* 評価関数による局面評価
* AI同士の自己対戦
* 対戦ログの自動保存
* 先攻・後攻を交互に変更した公平な評価

## 評価関数

評価関数では、連結している駒の数に応じてスコアを与えます。

例：

* 2連結の評価値
* 3連結の評価値

これらのパラメータを変更することで、AIの戦略や強さの変化を比較できます。

## 実験内容

以下の2種類のAIを用いて100試合の自己対戦を行いました。

### Three-focus AI

* 2連結：10点
* 3連結：50点

3連結を重視する設定です。

### Two-focus AI

* 2連結：30点
* 3連結：40点

2連結を積極的に評価する設定です。

また、先攻有利の影響を減らすため、各試合ごとに先攻・後攻を交代しています。
